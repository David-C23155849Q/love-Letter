# A Cinematic Web Experience

More than just a webpage—this project is an immersive digital experience.

Designed with motion, interactivity, and modern web technologies, this project blends animation, audio, and thoughtful UI design to create a website that feels alive. Every scroll, transition, and interaction is crafted to engage users and demonstrate what's possible with frontend development.

## Overview

Rather than presenting static content, this project focuses on storytelling through design. Smooth animations, interactive elements, and visual effects work together to deliver an engaging and memorable browsing experience.

---

## Features

### Immersive Landing Experience

An engaging introduction that welcomes visitors into the experience instead of immediately displaying content.

### Interactive Gallery

A dynamic gallery where images transition smoothly alongside meaningful quotes and a visual progress indicator.

### Integrated Music Player

A built-in music player powered by the Web Audio API, complete with a real-time audio visualizer that responds to the music.

### Custom Cursor

A fully customized cursor that enhances interaction and reinforces the overall visual identity of the project.

### AI Chat Interface

A modern floating chat component designed to integrate seamlessly with AI assistants or backend-powered chat systems.

### Modern Visual Effects

Carefully crafted glassmorphism, soft gradients, subtle film grain, glowing elements, and layered animations create a polished user experience.

## Running the Project

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Click **Enter Experience**.
4. Explore the animations, interactions, and audio features.

For the best experience, using headphones is recommended.
