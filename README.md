A Love Letter to the Web

Not just a webpage — an experience.

This project is a cinematic, interactive web page crafted with motion, music, and emotion.
Every scroll, hover, and beat is designed to feel something — not just show content.

✨ What Makes It Different?

💗 A story-driven landing
An intro screen that invites users in — because good experiences don’t start abruptly.

🎞 Living gallery, not a slideshow
Images fade, quotes breathe, and a progress bar quietly tells time.

🎧 Music that reacts
A built-in music player with an audio visualizer — the page literally moves with sound.

🖱 Cursor with personality
Because even the mouse deserves style.

💬 A floating AI chat corner
Minimal, modern, and ready to connect with automation or AI backends.

🌫 Grain, glow & glass
Subtle film grain, glassmorphism cards, soft gradients — tiny details, big vibe.

🛠 Built With

HTML + CSS (custom design, no frameworks)

Vanilla JavaScript (clean, purposeful logic)

GSAP & ScrollTrigger (smooth, cinematic motion)

Web Audio API (real-time music visualization)

No templates.
No UI kits.
Just handcrafted code.

🧪 How to Experience It

Put on headphones 🎧 (recommended)

Open index.html

Click Enter Experience

Let the page do the talking
